# Example Flow

This example demonstrates a simplified interaction cycle:

1. The system retrieves network state:

   * RTT = 50 ms
   * Throughput = 200 Mbps
   * Loss = 1%

2. The RL agent processes the state

3. The agent selects an action:

   * Increase pacing rate by 10%

4. The action is applied to the system

This represents one control step in the adaptive congestion control process.
