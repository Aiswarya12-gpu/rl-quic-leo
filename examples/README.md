# Examples

## Overview

This directory contains simple examples demonstrating how the reinforcement learning agent interacts with the simulation interface.

---

## Example: Interface Interaction

The example script shows how:

1. Network state is retrieved from the simulation interface
2. The RL agent processes the state
3. An action is applied to control the pacing rate

---

## Purpose

These examples are intended to illustrate the interaction workflow between the simulation environment and the RL agent using a PyBind11-based interface.

---

## Note

The examples are simplified demonstrations and are designed to support understanding of the system integration rather than represent full simulation execution.
